
# Data Refinement Project — Cafe Sales

## Objectif
Ce projet a pour objectif de transformer un dataset brut en un dataset propre et exploitable, en appliquant les principes de data quality et de data refinement.

## Dataset
Le dataset utilisé est "Cafe Sales – Dirty Data", représentant des transactions de ventes d’un café avec des données volontairement imparfaites.

## Structure du projet
- `DATA/RAW/` : dataset brut
- `DATA/PROCESSED/` : dataset nettoyé et final
- `NOTEBOOKS/` :
  - `01_EXPLORATION/` : exploration et identification des problèmes
  - `02_CLEANING/` : étapes de nettoyage
  - `03_TRANSFORMATION/` : transformations et vérifications finales
- `REPORTS/` : rapport final du projet
- `requirements.txt` : dépendances (non requises, travail réalisé via Google Sheets)

## Résultat
Le dataset final nettoyé est disponible dans le dossier `DATA/PROCESSED/` et est prêt pour une analyse fiable.
